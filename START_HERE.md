# 🎯 BẮT ĐẦU TỪ ĐÂY

Chào mừng! Bạn đã có tất cả những gì cần để tích hợp thông báo Zalo.

## 🚀 Bắt Đầu Nhanh (5 Phút)

### 1. Tạo Zalo OA
```
Truy cập: https://oa.zalo.me
→ Tạo OA mới
→ Lấy Access Token
→ Lấy OA ID
```

### 2. Cấu Hình
```bash
# Tạo file .env
ZALO_ACCESS_TOKEN=your_token_here
ZALO_OA_ID=your_oa_id_here
PORT=3000
```

### 3. Cài Đặt
```bash
npm install
```

### 4. Chạy
```bash
npm run dev
```

### 5. Test
```
Mở: http://localhost:3000
→ Đăng nhập Google
→ Trả lời bài test
→ Kiểm tra Zalo
```
